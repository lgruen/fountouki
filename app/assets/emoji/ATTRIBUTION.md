# Emoji art attribution

The PNG sprites in this directory are from **Twemoji**, the open-source emoji
art set (jdecked community fork).

- Source: https://github.com/jdecked/twemoji
- Graphics license: **CC-BY 4.0** (https://creativecommons.org/licenses/by/4.0/)
- Copyright 2020 Twitter, Inc and other contributors.

The graphics are used unmodified, fetched from the `assets/72x72/` set. Each
file is named by its emoji's Unicode codepoint(s) in lowercase hex, with the
U+FE0F variation selector stripped and multiple codepoints joined by `-`
(e.g. `1f436.png` = 🐶, `2708.png` = ✈️, `36-20e3.png` = 6️⃣).

Per CC-BY 4.0, attribution is required; this file provides it.
